## Support & Troubleshooting

### If APK won't install:
1. Check device compatibility (Android 7.0+)
2. Verify signature matches
3. Check storage space (100 MB needed)
4. Clear app cache if updating

### If app crashes on startup:
1. Check logcat: `adb logcat | grep FLUTTER`
2. Verify API endpoint is reachable
3. Check internet connectivity
4. Review crash logs on Play Console

### If Play Store upload fails:
1. Verify AAB build completes
2. Check all metadata fields filled
3. Ensure privacy policy URL is valid
4. Review content rating
